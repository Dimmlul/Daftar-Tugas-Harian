
# Laravel Project Setup

## 1. Clone Repository
```bash
git clone <https://github.com/Dimmlul/Daftar-Tugas-Harian.git>
cd <Daftar-Tugas-Harian>
```

## 2. Instalasi Composer
```bash
composer install
```

## 3. Instalasi NPM
```bash
npm install
```

## 4. Jalankan Development Server Untuk Load Tailwind
```bash
npm run dev
```
